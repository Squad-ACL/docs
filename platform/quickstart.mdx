This guide walks you through the essential setup steps—from creating your organization to inviting your team, getting your first server set up, and creating a permission group - all in minutes.

## Before you begin

- You will need to apply to become a partner in the Squad ACL Discord.
  - You can do so here: [https://squadacl.com/#partners](https://squadacl.com/#partners)
- All of your planned team members will need to be in here as well.
- It helps to have your server information ready, but is not needed.

## Steps

<Steps>
  <Step title="Become a Partner">
    You will need to create a partner request form within the Squad ACL Discord. This is required since ACL is still in Early Access and we want to gradually test our infrastructure.
  </Step>
  <Step title="Invite your team">
    Using the partner Discord channels, you can request for your team members to be added to Squad ACL. All they have to do is log in, complete the onboarding, and we can add them!
  </Step>
</Steps>

<Danger>
  Squad ACL is still in Early Access. Organizations will be accepted as we see fit. If your application is denied, you can still re-apply in the future once we have more spots available. We do limit the number of team members that an organization has, only to reduce server load during testing.
</Danger>
